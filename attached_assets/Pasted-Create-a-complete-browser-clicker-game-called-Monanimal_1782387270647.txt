- Web browser
- Mobile responsive (Android/iOS browsers)
- Single Page Application

Core Gameplay:
The player clicks on Monanimal to earn coins.

Currencies:
1. Coins (main currency)
2. MON (premium currency, reserved for future implementation)

Main Loop:
Click → Earn Coins → Buy Upgrades → Increase Income → Evolve Monanimal → Unlock New Areas
