You are a senior React, TypeScript and Tailwind game developer.

I am building a browser clicker game called "Monanimal Clicker".

The game is inspired by the Monad ecosystem.

I have already uploaded all graphical assets.

Your task is to completely replace placeholder graphics with my uploaded assets and build the game around them.

## Core Concept

The player clicks on Monanimal to earn Coins.

Coins are used to buy infrastructure upgrades.

Infrastructure upgrades increase CPS (Coins Per Second).

As the player progresses, Monanimal evolves visually.

The game should feel like upgrading a character rather than upgrading numbers.
